compressed_js_code

This html file download the necessary libraries to download and run a compressed js file.
The jszip library is used to download and unzip the file.
The "unzip_and_run.js" code is in charge of calling the jszip library and running the downloaded code.
Since the jszip library downloads and decompresses the code asynchronously, idealy, the all code will be self contained in the zip file.
The code runs all js files in the zip file.
